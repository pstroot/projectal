Easily add an Apple iPod Touch / iPhone icon, a favicon and more to your blog.

== Description ==

Easily add (by URL or upload) an Apple iPod Touch / iPhone icon, an iPod / iPhone boot image, a favicon and a feed image to your blog. And, should you wish, choose another favicon for your admin panel!

More info:

* [Blog Icons](http://yoast.com/wordpress/blog-icons/).
* Check out the other [Wordpress plugins](http://yoast.com/wordpress/) by the same author.

== Installation ==

Installation is easy:

* Download and unzip the plugin.
* Copy the folder to the plugins directory of your blog.
* Enable the plugin in your admin panel.
* An options panel will appear under Settings.
* Upload your images or enter their URLs.
